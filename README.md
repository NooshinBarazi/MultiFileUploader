# Multi File Uploader

This task is about make multiple file uploader via the Nextjs.

[Demo](https://multi-file-uploader-nine.vercel.app/)

# How to run project:

```bash
# Install deps
npm install --legacy-peer-deps

# Running in development mode
npm run dev

# Build it
npm run build

# Start for deployment
npm run start
```

## Stacks:

- NextJs
- Tailwindcss
- Vercel
